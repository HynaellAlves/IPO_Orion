## Ações Iniciais

Primeiro, execute o comando para iniciar o servidor de Desenvolvimento:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Adicione o endereço [http://localhost:3000](http://localhost:3000) na barra URL de seu navegador, confirme e observe o resultado.
